Hexo博客搭建

[toc]

概述

Hexo：一个快速、简洁且高效的博客框架。基于 Node.js 运行，使用 Markdown 解析文章。

Hexo官方文档

GitHub Pages：由 GitHub 官方提供的一种免费的静态站点托管服务，托管和发布静态网站页面。

GitHub Pages 官方文档

原理

本地编写MD，通过 Hexo 解析，渲染成具有主体样式的 HTML 静态网页，最后发布到 GitHub。

环境搭建

  1. 安装Git（使用的Mac安装Xcode后自动安装），使用 git --version 判断是否安装及版本

    下载地址：https://git-scm.com/downloads

  2. 安装Node.js，使用 node -v 判断是否安装及版本。npm 默认随 Node.js 安装。

    下载地址：http://nodejs.cn/download/

  3. 安装 Hexo，终端执行 $ npm install -g hexo-cli

完成后，文件夹内文件结构为

_config.yml：网站的配置信息，在此配置大部分参数。

package.json：应用程序的信息。

scaffolds：模板文件夹。新建文章时，Hexo 会根据 scaffold 来建立文件。

source：资源文件夹。开头命名为_的文件/文件夹和隐藏的文件会被忽略。

Themes：主题文件夹。Hexo会根据主题来生成静态页面。
